# LangChain Summarizer: YouTube & Website Content

This project is a **Streamlit app** that summarizes content from YouTube videos or websites using **LangChain** and **ChatGroq**. Users can provide a URL, and the app will extract the content and generate a concise 300-word summary.

---

## Features

- Summarizes text content from **YouTube videos** or **web pages**.
- Uses **LangChain chains** for summarization.
- Powered by **ChatGroq** for AI text generation.
- Provides input validation for URLs and API keys.
- Interactive Streamlit interface.
